# C# Topics Index

Each example is self-contained and can be run with `dotnet run -- <topic>`.

| Topic | File | What it demonstrates |
| --- | --- | --- |
| Basics | `Examples/BasicsExample.cs` | Variables, data types, interpolation, arithmetic |
| Control flow | `Examples/ControlFlowExample.cs` | Conditions, ternary expressions, `for` loops |
| Methods | `Examples/MethodsExample.cs` | Parameters, return values, `params` |
| Strings | `Examples/StringsExample.cs` | String methods and interpolation |
| Collections | `Examples/CollectionsExample.cs` | `List<T>`, adding values, indexing |
| OOP | `Examples/ObjectOrientedProgrammingExample.cs` | Classes, properties, encapsulation |
| Exception handling | `Examples/ExceptionHandlingExample.cs` | Safe parsing with `TryParse` |

## Running examples

```bash
dotnet run -- basics
dotnet run -- oop
dotnet run -- --all
```

Use `dotnet run -- --list` to display the available topic names in the console.
